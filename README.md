# Time Travel Message

Send messages to your future self. Receive messages from your past.

## Features

### Send to Future
- Write messages to your future self
- Preset delivery times: Tomorrow, 1 Week, 1 Month, 1 Year
- Custom date selection
- Message preview before sending

### Time Capsules
- View scheduled messages
- See countdown to delivery
- Edit or delete pending messages

### From the Past
- Receive messages when their time arrives
- Reflection on past thoughts and goals
- Delete after reading

## Usage

1. Open `index.html` in a browser
2. Write a message to your future self
3. Choose when to receive it
4. Come back later to find your message waiting

## Why Use This?

- Set reminders for future goals
- Write encouragement for hard times
- Document predictions to review later
- Create personal time capsules
- Reflect on personal growth

## Tech

- Local storage (messages stay on your device)
- No server - completely private
- Works offline
- Background notifications (if permitted)
